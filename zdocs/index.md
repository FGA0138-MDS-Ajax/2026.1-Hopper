# HopLife

Bem-vindo à documentação oficial do sistema **HopLife**.

O HopLife é uma aplicação web desenvolvida pelo Grupo Hopper para a disciplina de Métodos de Desenvolvimento de Software (MDS), com foco em acessibilidade, inclusão digital e gamificação para usuários idosos.

## Objetivo do Projeto

O sistema tem como objetivo auxiliar usuários no gerenciamento de metas pessoais e hábitos cotidianos de maneira simples, intuitiva e acessível.

Entre os principais recursos estão:

- Gerenciamento de metas pessoais
- Acompanhamento de progresso
- Interface acessível para idosos
- Sistema de gamificação baseado no framework Octalysis
- Recursos de acessibilidade visual e motora

## Tecnologias Utilizadas

- Python
- Django
- Bootstrap
- SQLite
- Docker
- Keycloak
- SonarQube
