# HopLife — Ambiente de Desenvolvimento

Este é o repositório central de desenvolvimento do **HopLife**. Esta branch contém o código-fonte do backend, configurações de ambiente e os containers estruturados para rodar a aplicação localmente.

---

## Pré-requisitos

Antes de iniciar, certifique-se de ter instalado em sua máquina:
* **Docker** e **Docker Compose**
* **Python 3.x**
* **Git**

---

## Como Configurar e Rodar o Projeto (Windows)

Basta instalar as depedências do requirement.txt para configurar seu ambiente de desenvolvimento.

Para criar seu ambiente docker apenas digite:

docker build -t nome-da-imagem .
docker-compose up

### Passo Único:
Execute o arquivo .bat que ele vai fazer tudo o que precisa ser feito.
